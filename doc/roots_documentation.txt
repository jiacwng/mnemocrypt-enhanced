Data transfer:
    MOV - move data
    XCH - exchange data
    IN - transfer data from input
    OUT - transfer data to output
    LEA - load effective address
    LD - load
    LES - load far pointer
    LAHF - Load Status Flags Into AH Register
    SAHF - Store AH Into Flags
    UNPCK - unpack
    INSR - insert Byte/Dword/Qword
    INSER - insert
    XTR - extract (for FXTRACT)
    EXTR - extract
    ST - store
    RD - read
    WR - write
    BROADCAST - load with broadcast
    SLDT - store local descriptor table register (DTR)
    SGDT - store global DTR
    SIDT - store interrupt DTR
    LLDT - load local DTR
    LGDT - load global DTR
    LIDT - load interrupt DTR
    PREFETCH - prefetch
    BLD - load binary-coded decimal (BCD)
    BST - store BCD and pop
    PACK - pack
    ALIGN - align data
    XLAT - table lookup translation
    MOVSX - move with sign extension

    PREM - partial remainder

    SAD - sum of absolute differences
    CHS - change sign
    ROUND - round

    PTAN - partial tangent
